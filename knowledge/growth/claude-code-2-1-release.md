- **Configure response language** — eg. Japanese, Spanish

### Skills & Agents
- **Add hooks directly to agents & skills frontmatter**
- **Forked context** for skills
- **Hot reload** for skills
- **Custom agent support**
- **Invoke skills with /** (slash command)

- **Agents no longer stop when you deny a tool use**
- **Wildcard support for tool permissions** — eg. `Bash(*-h*)`
- **/teleport your session** to claude.ai/code
