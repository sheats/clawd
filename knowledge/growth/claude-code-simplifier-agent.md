## What It Is

The Claude Code team open-sourced the code-simplifier agent they use internally.

## Installation

From terminal:
```bash
claude plugin install code-simplifier
```

Or from within a session:
```
/plugin marketplace update claude-plugins-official
/plugin install code-simplifier
```

## When to Use

- At the end of a long coding session
- To clean up complex PRs
- To simplify convoluted code

## What It Does

Analyzes and simplifies code, making it more readable and maintainable.
