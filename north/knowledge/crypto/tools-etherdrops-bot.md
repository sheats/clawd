# Etherdrops Bot

URL: https://x.com/etherdrops_bot
Telegram: @etherdrops_bot

## What It Is
Telegram bot for tracking crypto wallets and getting alerts on transactions.

## Features
- Wallet tracking across multiple chains
- Transaction alerts
- Token transfer notifications
- Portfolio tracking via Telegram

## Use Cases
- Monitor your own wallets for activity
- Track whale/smart money wallets
- Get notified of large transfers
- Alert on specific token movements
